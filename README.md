# XINSHENGAN Traffic Safety

Cinematic multilingual traffic-safety product website featuring:

- Six-language interface with Arabic RTL
- Localized product images and cinematic intro video
- Real GLB / Three.js traffic bollard showroom
- Product filtering, detail drawer, and comparison
- Specification configurator and copyable inquiry summary
- GitHub Pages automated deployment

## Public sites

- Accio Live Site: https://xinshengan-safety.site.accio.ai
- GitHub Pages: https://1pp44.github.io/xinshengan-traffic-safety/

## Local development

```bash
npm ci
npm run dev
```

## Build

Root-domain build:

```bash
npm run build
```

GitHub Pages subpath build:

```bash
npm run build -- --mode github
```

The 3D model is a visual presentation asset and is not CAD or manufacturing data.
