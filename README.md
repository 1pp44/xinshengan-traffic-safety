# 鑫盛安交通设施 (XINSHENGAN Traffic Safety)

电影感多语言交通安全产品独立站，具有以下特性：

- 支持六种语言界面，包含阿拉伯语 RTL 布局
- 本地化产品图片与电影感入场视频
- 基于 Real GLB / Three.js 的交通警示柱 3D 展厅
- 产品过滤、详情抽屉与对比功能
- 规格配置器与可复制的询盘摘要
- GitHub Pages 自动部署

## 公开地址

- Accio 实时站点: https://xinshengan-safety.site.accio.ai
- GitHub Pages: https://1pp44.github.io/xinshengan-traffic-safety/

## 本地开发

```bash
npm ci
npm run dev
```

## 构建

根域名构建：

```bash
```bash
npm run build
```

GitHub Pages 子路径构建：

```bash
npm run build -- --mode github
```

注：3D 模型仅为视觉展示资产，并非 CAD 或生产制造数据。
